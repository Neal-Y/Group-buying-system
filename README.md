# Installation && Setup
1. Clone the repository and into to directory.
    ```
    git clone git@github.com:jonyig/devmentor-BE103-golang.git && cd devmentor-BE103-golang
    ```

2. Start the container.
   ```sh
   docker-compose up -d 
   ```

3. Check API document
    ```
   localhots:8000
   ```
